# Diabet Checker Agent Scenario

Diabet Checker — учебный сквозной проект: небольшой сервис для самоконтроля уровня глюкозы и ежедневных действий, которые помогают пользователю следить за своим состоянием.

## Общая цель проекта

Создать минимальную рабочую основу продукта, в котором пользователь может видеть последнее измерение глюкозы, список действий на сегодня и ближайшее напоминание.

Проект не ставит диагнозы, не назначает лечение и не заменяет консультацию врача. Его задача — помогать пользователю фиксировать данные и соблюдать регулярные действия по самоконтролю.

## Фокус первой итерации

Первая итерация ограничена одной пользовательской зоной: личной панелью контроля глюкозы на сегодня.

В неё входят:

- последнее измерение глюкозы;
- статус уровня глюкозы в простом виде;
- список действий на сегодня;
- отметка выполненных действий;
- ближайшее напоминание;
- простая карточка профиля с именем и почтой;
- предупреждение о том, что приложение не заменяет врача.

## Почему выбран такой срез

Этот срез достаточно маленький для одной рабочей сессии с агентом, но при этом похож на реальную первую итерацию продукта. Он не требует сложной медицинской логики, интеграций с глюкометрами, авторизации, уведомлений или полноценной базы данных.

Агент может сосредоточиться на структуре проекта, понятной пользовательской панели, моковых данных и безопасных ограничениях.

## Артефакты для работы агента

В репозитории подготовлены:

- `AGENTS.md` — постоянные инструкции для агента;
- `docs/iteration-1-plan.md` — пошаговый план первой итерации;
- `docs/progress-control.md` — точки контроля, по которым проверяется, что агент не ушёл в сторону.

## Учебный срез модуля

В рамках этого задания проект показывает, как организовать работу агента: задать цель, ограничить медицински чувствительный контекст, разложить работу на шаги и заранее определить проверки прогресса.

Главный результат — не полноценное медицинское приложение, а рабочий сценарий, по которому агент сможет безопасно подготовить первую реалистичную итерацию продукта.
